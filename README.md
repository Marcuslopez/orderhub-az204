# 🚀 OrderHub - AZ-204 Training Project

## 📌 Descripción

Este repositorio contiene la implementación de prácticas y conceptos adquiridos durante la preparación para la certificación **AZ-204: Developing Solutions for Microsoft Azure**.

El proyecto está enfocado en el desarrollo de aplicaciones cloud modernas utilizando servicios de Azure, aplicando buenas prácticas de arquitectura, seguridad y automatización.

---

## 🎯 Objetivos

* Diseñar y desarrollar aplicaciones escalables en la nube
* Implementar APIs modernas y servicios serverless
* Integrar múltiples servicios de Azure
* Aplicar seguridad y gestión de identidades
* Automatizar despliegues mediante CI/CD
* Trabajar con contenedores y orquestación

---

## 🧠 Conocimientos Aplicados (AZ-204)

### 🔹 Desarrollo de APIs y servicios

* ASP.NET Core Web API
* Diseño RESTful
* Manejo de autenticación y autorización (JWT / Azure AD)

### 🔹 Computación Serverless

* Azure Functions
* Event-driven architecture
* Integración con triggers (HTTP, queues, etc.)

### 🔹 Almacenamiento en Azure

* Azure Blob Storage
* Azure Table / Cosmos DB (NoSQL)
* Manejo de datos estructurados y no estructurados

### 🔹 Integración de servicios

* Azure Service Bus / Queues
* Event Grid / Event Hub
* Comunicación entre microservicios

### 🔹 Seguridad

* Azure Active Directory (AAD)
* Role-Based Access Control (RBAC)
* Manejo seguro de secretos (Key Vault)

### 🔹 DevOps & CI/CD ⚙️

* Automatización de despliegues
* Pipelines de integración y entrega continua (CI/CD)
* Uso de herramientas como GitHub Actions o Azure DevOps
* Versionamiento y control de código

### 🔹 Contenedores y Kubernetes 🐳

* Creación y despliegue de contenedores (Docker)
* Uso de Azure Container Instances / App Services
* Introducción a Kubernetes (AKS) para orquestación
* Despliegue de aplicaciones escalables en contenedores

### 🔹 Monitoreo y rendimiento 📊

* Azure Monitor
* Application Insights
* Logging y diagnóstico de aplicaciones
